怎么拿到 API Key？

这份教程写给完全不了解计算机的朋友。请按顺序一步步做。推荐使用 DeepSeek（深度求索）——本软件默认就是它，价格相对友好，步骤也简单。

先弄懂三个词（30 秒）

  • API Key：可以理解成一把「私人钥匙」。软件用它向 DeepSeek 请求写小说。钥匙是保密的，不要发给陌生人，也不要发到微信群、朋友圈。
  • 充值 / 余额：用 AI 写字一般要花一点点钱（通常几毛到几块钱就能试很多次）。没钱时钥匙也能创建，但写不出来。
  • 粘贴：复制文字后，在输入框里点一下，再按 Ctrl + V（苹果电脑是 Command + V）。

1打开 DeepSeek 开放平台

  1. 用电脑打开浏览器（Chrome、Edge、Safari 都可以）。
  2. 在地址栏输入下面网址，然后按回车：
    https://platform.deepseek.com/
  3. 如果网页是英文，先不用慌，后面有中文按钮的地方照着点即可。

如果打不开：检查网络，或换一个浏览器再试。公司/学校网络有时会拦截，可换手机热点试试。

2注册或登录账号

  1. 在页面上找到 Sign up / 注册 或 Log in / 登录。
  2. 按提示用手机号或邮箱完成验证（会收到验证码短信/邮件）。
  3. 设置一个密码，并记在安全的地方（不要用生日这种太简单的）。
  4. 登录成功后，你会进入「控制台 / Dashboard」一类的管理页面。

3进入 API Key 页面并创建钥匙

  1. 登录后，在左侧或顶部菜单里找到类似这些字样的入口之一： API Keys、API 密钥、密钥管理。
  2. 也可以直接打开： https://platform.deepseek.com/api_keys
  3. 点击 Create API Key / 创建 API Key / 创建新密钥。
  4. 给这把钥匙起个名字，例如：aiwriter（随便起，方便你自己认出就行）。
  5. 创建成功后，网页会显示一长串以 sk- 开头的字符——那就是 API Key。

非常重要：完整钥匙往往只显示一次。请立刻复制保存到备忘录或记事本。关掉页面后，可能再也看不到完整内容，只能删掉重建。

4给账户充一点钱（才能真正写小说）

  1. 在平台里找到 Billing / 充值 / 余额 / Top up 等入口。
  2. 按提示充值（金额按页面说明；新手一般先充少量试用不影响）。
  3. 充值成功后，余额里应能看到金额。

若未充值：创建了 Key 也可能「连得上但写不出」或提示额度不足。这是正常现象，不是软件坏了。

5把钥匙填进 AIWriter

  1. 打开 AIWriter 网站，点顶部的 设置。
  2. 「服务商预设」请选择 DeepSeek（默认）。
    （一般已经选好了，不用改。）
  3. 「API URI」一般会自动变成：
    https://api.deepseek.com/v1
    请不要随便改。
  4. 「模型名称」推荐保持：
    deepseek-v4-flash
  5. 在 API Key 输入框里，粘贴你刚才复制的那串 sk-...。
  6. 回到顶部点 写作，在右侧写一点要求，再点 开始写作。

如果正文开始一个字一个字往外蹦，就说明成功了。恭喜！

常见问题

  • 提示鉴权失败 / 401？
    Key 复制少了字符、多了空格，或 Key 已被删除。重新创建一把再粘贴。
  • 提示额度不足 / 429？
    余额不够或请求太频繁。去平台充值，稍后再试。
  • 提示网络或 CORS？
    这是浏览器安全限制。可换网络、稍后再试；若仍不行，需要懂技术的朋友帮你配置「允许跨域的网关」（这超出零基础范围，可找人协助）。
  • 钥匙泄露了怎么办？
    立刻回 API Keys 页面删除旧钥匙，再新建一把，并更新 AIWriter 设置里的 Key。
  • 想用别的厂家（OpenAI、通义、硅基流动）？
    流程类似：去对应官网注册 → 找到 API Key 页面 → 创建 → 充值 → 在 AIWriter 设置里换「服务商预设」并粘贴 Key。默认仍建议先用 DeepSeek 练熟。

安全提醒（请务必阅读）

  • API Key 等于花钱权限，不要截图发群、不要交给陌生人「帮你调试」。
  • AIWriter 把 Key 存在你自己的浏览器里，不会发到本软件自己的服务器。
  • 用完公共电脑，请到设置里点「清除密钥」或「清除本机全部数据」。
